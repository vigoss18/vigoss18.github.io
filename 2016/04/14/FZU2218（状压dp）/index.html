FZU2218（状压dp）

发表于 2016-04-14 | 分类于 ACM |

DP好叼啊……我暴力了半天才发现自己是错的，用dp竟然那么短，思路还那么清晰……脸都肿了……

题目链接：http://acm.fzu.edu.cn/problem.php?pid=2218

题目意思是给你一个长度为n的字符串，有k个不同的字符，要你选出两个子串，要求这两个子串互不包含另一个子串出现过的任意一个字符，同时使得长度之积最大。

一开始我想着用双指针搞一搞，结果死活过不了，对拍了一些数据后才发现，我这样搞的话，必须把所有的字符都用进去，然而其实有些字符不选，其长度更长……

所以正确的思路是先$n^2$枚举区间，用dp[i]来二进制记录第k位是否使用了。这样得到的dp数组就是所有为1的位置都使用上的时候最长的长度了。但是这样来枚举的话，因为可以有一些字符不选，所以复杂度很高，就要考虑用状压dp搞一搞了。可以处理把dp[i]处理成用其中某些是1的字符的最大长度，接下来就可以直接枚举了。

真的好叼啊……

#include<string>
#include<iostream>
#include<cstring>
#include<algorithm>
#include<cstdio>
#include<map>
#include<cmath>
#include<vector>
#define ll long long
using namespace std;

int dp[1 << 18];

int main()
{
    int t;
    cin >> t;
    while(t--)
    {
        int n,k;
        string s;
        cin >> n >> k >> s;
        memset(dp,0,sizeof(dp));
        for(int i = 0;i < n;i++)
        {
            int S = 0;
            for(int j = i;j < n;j++)
            {
                S |= (1 << (s[j] - 'a'));
                dp[S] = max(dp[S],j - i + 1); 
            }
        }
        for(int i = 0;i < 1 << k;i++)
        {
            for(int j = 0;j < k;j++)
            {
                if(i >> j & 1)
                    dp[i] = max(dp[i],dp[i ^ (1 << j)]);
            }
        }
        int ans = 0;
        for(int i = 0;i < 1 << k;i++)
        {
            ans = max(ans,dp[i] * dp[(1 << k) - 1 ^ i]);
        }
        cout << ans << endl;
    }
}
