虽然自己的第一次面试还是蛮顺利的，但是谁知道以后还会不会要面试呢，还是把自己总结起来的面试题目记录一下吧。

更新于2018年11月

C/C++

extern C的作用：
由于C++支持函数重载，但是C并不支持，所有用extern C来表明这个函数或者变量是用C语言来编译和连接的，这样做可以解决名字匹配问题，实现C++和C的混合变成。

const的作用：
1、用于定义const常量。
2、用于修饰函数的参数和返回值等，被修饰的东西都受到了强制保护，能预防意外的发生，提高程序的健壮性。
3、const常量赋值时必须被初始化。

const和define的区别：
1、const有数据类型，而define没有。
2、const能由编译器进行安全检查，但是define只是单纯的字符串替换，没有安全性。
3、集成化的调试工具能对const常量进行调试，但是不能对宏常量进行调试。

在const成员函数中，用mutable修饰成员变量名之后就可以修改类的成员变量了。

sizeof是一个运算符，sizeof()，括号内的内容是被替代类型，如：
int a = 8;sizeof(a)，这个a会被替代成int。用来计算一个类型的字节大小。
如果sizeof里面是指针，返回的永远是4。

内联函数和宏的差别：
1、内联函数在编译的时候可以直接被镶嵌到目标代码里去，宏只是简单的替换。
2、内联函数要做参数安全检查，宏不能。
3、内联函数是以增加空间消耗为代价的。

内联函数的适用：
1、一个函数不断被重复调用。
2、函数简短，且不包含for、while、switch等语句。

指针和引用的区别：
1、非空区别。任何情况下引用都不可能指向空值。
2、合法性区别。由于指针可能为空值，所以需要被测试，而引用则不需要测试合法性。
3、可修改区别。指针可以被赋值指向不同的对象，但是引用则只能指向在初始化时被指定的对象，但是对象的内容是可以改变的。
4、应用区别。综合上面三点区别，指针通常在考虑到可能会不指向任何对象的可能，或者可能会指向不同对象的可能时使用，其他时刻应该使用引用。

引用传递，修改的结果直接影响实参。

Socket编程学习————客户端和服务端的编程

发表于 2017-03-27 | 分类于 socket |

由于想学习一下网络爬虫以及拿到的第一份offer涉及到socket编程，这几天初步学习一下Windows环境下的socket编程。

Socket

socket的本质是一种编程接口(API),它是应用层和TCP/IP协议族通信的中介。如果说HTTP是车的外壳，提供了封装或者显示数据的具体形式，那么socket就是发动机，给其提供了网络通信的功能。
在一台连接了互联网的主机上，运行着多个软件，每个软件打开一个socket，并且绑定到一个端口上去，不同的端口对应不同的服务软件。正如socket的英文原意一样，socket就像一个多孔插座，每个插座有不同的编号，不同的插座提供不同的服务。

ICPC沈阳

发表于 2016-11-19 | 分类于 ACM-现场赛 |

距离沈阳赛已经一个月了才提起劲来写后记……懒癌无限大の我……

沈阳是我这两年去过最远的赛区了，火车竟然要30h+，为了这趟远门特地又买了一个充电宝以备不时之需，其实一开始我们队是不怎么想去沈阳的，因为教练和我们说这场是拿牌了才能报销全部的钱否则只能报销一半，毛估估如果不获奖就要自己垫1k在里面了……但是教练一副信誓旦旦的样子：沈阳是全部赛区里最弱的了。……我当时为了会信了教练的鬼话啊！？？

Day0：

因为要30h+的缘故，为了能赶上热身赛的我们只能买了凌晨的车票了，结果尼玛宝树早早就把我们拉过去了，买了泡面和矿泉水还有好久……于是我们决定去火车站旁边的网吧凑活几小时。虽然有预想过火车站旁边的网吧肯定不会好到哪里去但是没想到这么破……简直无力吐槽，我家08年买的电脑都比这个配置好。点开b站几小时就过去了……临近1点了，离火车开车还有一个小时不到点，我们出来吃了顿宵夜（好像是吃了？……）然后过安检候车，第二次见到凌晨的火车站了，第一次是从河南回来的时候，那时候打了我在国赛上的第一块铁，此刻去年的打铁瞬间仿佛又在眼前闪过……完了啊这是卡密萨马对我的暗示啊！

残念无比地上了火车，精疲力尽的我睡了过去……
在火车上的一天就跳过去了……已经忘记发生了什么了……估计什么都没有发生吧。

Day1：

早上八点到了沈阳北，然后按照打车到了东北大学附近的“门”。下了车的我们是懵逼的……明明地图显示的北门，然而在我们面前的竟然是一个商场和一片施工工地，问了几个大叔才找到正确的门……进到学校打开ingress就发现有好多点啊（为什么除了我校都有那么多点……）刚想炸炸炸就被队友拉去报道了，报道完觉得还早于是就出学校去找宾馆了。路上我和队友说我们顺路买个早饭吧，于是找了路边一家买鸡蛋灌饼的店，一问竟然只要四块五，加了香肠鸡蛋还有一片肉……这绝对是我吃过最划算但却最好吃的饼了……

到了宾馆扔下行李，然后云泽他们来找我，L的发型真帅，原来中分也能那么帅……果然还是长相问题吗。然后我们一起去吃中饭，然而云泽和L并没有饭票于是我们暂时分开了。中饭是自助餐，相当好吃……尤其是米饭，不愧是东北。。。真的好好吃啊。一脸满足.jpg

热身赛，第一题好像是猜数字，队友疯狂大喊，说就是建校年份，1A，后面的题目是啥已经忘记了……反正每次热身赛的题目都是，看着极限数据会卡，然而暴力出奇迹的题目……只能说并没有败人品吧。热身完又回去吃了一顿一模一样的晚饭，依旧一脸满足……

ICPC青岛

发表于 2016-11-17 | 分类于 ACM-现场赛 |

不知道应该怀揣着怎么样的心情来写这篇退役之行的后记，感觉比赛本身对我来说已经不那么重要了，重要的时候遇到了哪些风景，和哪些人。

Day 0

因为早上八点的火车，所以大家六点就“早早地”起床了，火急火燎地穿完衣服就冲出寝室上了预约好的顺风车。一路无言。赶到火车站之后大家在大娘水饺吃了一顿蒸饺，20左右，还是吃的挺饱的。想吐槽一下那个牛肉粉丝汤啊……尼玛还真的只有牛肉和粉丝。。一点调味料都木有。顺利地过安检，上了已经坐过好多次的绿皮火车，逛吃逛吃……一天就这么睡过去了，睡在我旁边的竟然是个可爱的小姐姐……可惜全程背对我，残念……

Day 1

早上五点半就被乘务员姐姐叫醒了，毕竟6点就要到青岛北站了。下了火车之后大家合计了一下，决定做的士，六个人分别上了两辆的士，上去的那一刻我注意到这俩的士的计价器别藏在脚边就有种不祥的预感了……虽然有注意计价器的跳动，但好像并没有发现什么异常，结果到石油大学一看，138+20元……尼玛啊，说好的只要7、80呢，真坑，就这么宰外地人么？我要了发票……不过想来也并没有什么卵用，均摊到每个人也就50……不了了之了。

的士带我们到的时候石油大学的南门，正好面对着大海，又是一个出校门就能看到海的大学啊……真棒。无聊地打开ingress，发现这个学校竟然有好多好多的点，南门这几个已经被doc老师占了，还好脚的等级不高，兴奋地炸炸炸，迷之败人品……这可是出题人占的点啊喂！不过这都阻止不了我这个四级萌新了！和老师汇合后在校园里一阵逛，我一直在兴奋地摸点连link，超多距离超近的点！我要做多重升级！早饭又吃了一顿水饺，10块钱超大一盘，实惠又好吃！然后我就溜出队伍了继续玩ingress去了，顺利升到5级，石油大的大佬想要面基我，吓得我赶紧跑路去宾馆和队友汇合了……

中饭是番茄炒蛋+番茄蛋花汤+小鸡腿两个，不得不吐槽真的不好吃……还好早饭吃得比较晚。见到了qwb，一阵乱摸，不知道是不是最后一次见到他了。体育馆很大，但是桌子好小啊，我们三个人挤在一张桌子上，手都放不开。热身赛这次非常败人品……竟然被我们AK了，然后队友就非常装逼地说“我们是金牌队啊”，我都看不下去了……
