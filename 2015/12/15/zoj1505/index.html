一道傻逼题竟然卡了我半天……我真是难受啊。zoj1505

题意是8*8的棋盘上有四个棋子，给你初状态和末状态，问你能否在8步之内从初状态转化到末状态，转化的方式是普通走棋子+跳棋。

之前我做的时候貌似先想到用ll来存整个棋盘的信息，但是当时好像遇到了问题，询问过太虚之后，太虚给我指出可以在int的范围内存下来，即一颗棋子占据6位，于是我今天首先开始写的是用太虚说的办法写的，但是写着写着，我发现这样的话，如果我存储棋子的顺序不同，这些状态值就会被我判断成是不同的状态，然后我尝试着排序，不过还是越写越乱，于是我决定还是用我一开始想到的方法，用ll来存储整个棋盘的状态，然后每次枚举可行的走法，写完一次之后过了样例，交了一发，wa，然后开始各种修改，我来总结下我遇到的问题。

首先我竟然从头至尾的三个小时里面都写错了搜索的层数，结果把步数控制在10步以内，导致了我其实存错了棋盘信息但是却返回了yes，让我有了一种我写的搜索是对的的错觉，并且让我mle了两次。其次我一开始写的时候，没有把一些重复使用的功能封装成函数，然后我朋友就一个劲地吐槽我的风格……不过后来我缩减了，也减小了debug的难度。最后，我写到最后发现我的bug竟然是使用了错误的变量，我设f为当前的棋盘状态，我异或掉要移动的棋子之后，添加新的棋子时，竟还是在用f来惊醒异或，导致我的棋盘变成了5颗棋子甚至更多。这才是我后来一直wa的原因。

真的是心细才是成功的关键啊，2015年12月15日01:28:46，我也该睡觉了……不然明天又要起不来了

#include <bits/stdc++.h>
#define ll long long
using namespace std;

ll st,ed;

set<ll>st1,st2;

int dx[] = {-1,-2,1,2,0,0,0,0},dy[] = {0,0,0,0,-1,-2,1,2};

bool check(int x,int y)
{
    if(x < 0 || x > 7 || y < 0 || y > 7)
        return false;
    return true;
}

void wtf(ll n)
{
    for(int i = 0;i < 64;i++)
    {
        if(n & (1ll << i))
        {
            cout << i / 8 << " " << i % 8 << " ";
        }
    }
    cout << endl;
}

void bfs1()
{
    queue<pair<ll,int> > q1;
    q1.push(make_pair(st,0));
    st1.insert(st);
    while(!q1.empty() && q1.front().second < 4)
    {
        ll f = q1.front().first;
        int s = q1.front().second;
        int mp[8][8],x[4],y[4],cnt = 0;
        memset(mp,0,sizeof(mp));
        for(int i = 0;i < 64;i++)
        {
            if(f & (1ll << i))
            {
                mp[i / 8][i % 8] = 1;
                x[cnt] = i / 8,y[cnt++] = i % 8;
            }
        }
        for(int i = 0;i < cnt;i++)
        {
            for(int j = 0;j < 8;j += 2)
            {
                ll now = f ^ (1ll << (x[i] * 8 + y[i]));
                if(check(x[i] + dx[j],y[i] + dy[j]) && !mp[x[i] + dx[j]][y[i] + dy[j]])
                {
                    //cout << i << endl;
                    now = now ^ (1ll  << ((x[i] + dx[j]) * 8 + y[i] + dy[j]));
                    if(!st1.count(now))
                    {
                        st1.insert(now);
                        q1.push(make_pair(now,s + 1));
                    }
                }
                else if(check(x[i] + dx[j + 1],y[i] + dy[j + 1]) && !mp[x[i] + dx[j + 1]][y[i] + dy[j + 1]])
                {
                    //cout << i << endl;
                    now = now ^ (1ll  << ((x[i] + dx[j + 1]) * 8 + y[i] + dy[j + 1]));
                    if(!st1.count(now))
                    {
                        st1.insert(now);
                        q1.push(make_pair(now,s + 1));
                        //cout << now << endl;
                       // system("pause");
                    }
                }
            }
        }
        q1.pop();
    }
}
