A·Uncowed Forces

就是给你cf的rating的计算公式让你算一下某人一场cf下来的分数。因为要求所有的计算都不能有精度误差，全部用double来写。ORZZ当时改来改去还是wa在test6上，好郁闷。

#include <bits/stdc++.h>
#define ll long long
#define eps 1e-6
#define MOD 10007
using namespace std;

int main()
{
     double m[5],w[5];
     double so[5] = {500,1000,1500,2000,2500};
     for(int i = 0;i < 5;i++)
        cin >> m[i];
    for(int i = 0;i < 5;i++)
        cin >> w[i];
    double hs,hu;
    cin >> hs >> hu;
    double s = 0;
    for(int i = 0;i < 5;i++)
    {
        s += max(0.3 * so[i],(1.0 - m[i] * 1.0 / 250) * so[i] - 50 * w[i]);
    }
    s -= hu * 50 - hs * 100;
    printf("%.0f\n",s);
    return 0;
}

B·More Cowbell

给你n个物品和m个箱子，每个物品的重量为si，求最小的箱子承重量使得箱子能装下全部的物品，当时我漏看了一个箱子最多只能装两个物品，结果惨WA，用优先队列水过。

#include <bits/stdc++.h>
#define ll long long
#define eps 1e-6
#define MOD 10007
using namespace std;

struct cmp
{
    bool operator()(const pair<ll,int> a,const pair<ll,int> b)
    {
        if(a.second != b.second)
            return a.second > b.second;
        return a.first > b.first;
    }
};

int main()
{
     ll n,m,a[100005];
     cin >> n >> m;
     for(int i = 0;i < n;i++)
     {
        cin >> a[i];
     }
     priority_queue<pair<ll,int>,vector<pair<ll,int> > ,cmp>q;
     for(int i = n -1;i >= 0;i--)
     {
        if(q.size() < m)
        {
            q.push(make_pair(a[i],1));
        }
        else
        {
            ll u = q.top().first;
            //cout << u << endl;
            q.pop();
            q.push(make_pair(u + a[i],2));
        }
        //cout << q.top() << endl;
     }
     ll st = 0;
     while(!q.empty())
     {
        st = max(q.top().first,st);
        q.pop();
     }
     cout << st << endl;
    return 0;
}

C·Alternative Thinking

给你一个长度为n的01串，反转其中连续的一段子序列之后求最长的{1,0,1,0……}串可以是不连续的。
当时猜想如果没有变化前，长度为n - 2及一下的话，反转一下一定能使其增加2，否则就使其长度变为n，跑了一些小数据对拍了一下是过的，于是交，1Y，哈哈，YY大法好。

E·Lieges of Legendre

给你n堆石子，每次你可以有以下两种操作：
1.选一堆石子，拿走其中一块石子。
2.选择一堆2*x的石子，把他分成k堆x块石子。

今天刚学的sg啊。要按照k的奇偶性来分类讨论。

##首先需要记住的是sg值的含义！！SG[x] = mex{x}，即所有x可能产生的后继状态的SG值中，不存在这些值中最小的非负整数。 这个很重要。

设x为当前选择石子堆的石子数，当k为偶数时，如果u是偶数，就能转移到u-1或者k*x，因为k为偶数，那么这种情况下的异或和为0，如果u是奇数，那么就只能转移到状态1，于是u为偶数时的sg值为1，u为奇数时的sg值为0，当然，u是2的话，sg值是2。

当k为奇数时，2，3，4需要特判，当u为偶数时，你分成k堆后的状态值为x，一定不为0，那么所有u为奇数的状态的后续状态的sg值一定不含0，于是u为奇数时，sg值一定是0，当u为偶数时，你可以一直分块直到u不是奇数，sg值是1和2不断交替的，那你只要再判断一下到底分块了几次就可以了，同时可以把3的状态处理进来当成一次分块。
