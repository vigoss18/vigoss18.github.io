dp初步学习1

发表于 2015-11-08 | 分类于 ACM-dp |

今天初步学习了一下dp问题，着重研究了一下多重背包。

先把大牛写的背包九讲供起来

先做了51NOD上最简单而且经典的一道dp题，编辑距离。
其实一开始我是不懂为什么我一眼看上去像是暴力的题目要用dp来做，后来分析了一下，觉得还是很有道理的。题目是给你两个字符串，你可以用1.删除一个字符,2.替换一个字符,3.插入一个字符。

那么就用dp[i][j]来表示第一个字符串的i位置来和第二个字符串的j位置配对，如果配对成功，则dp[i][j] = dp[i - 1][j - 1]。

替换：替换就是替换一个字符来满足两者匹配，dp[i][j] = dp[i - 1][j - 1] + 1;

删除：删除就是把一位往后推1，即dp[i][j] = min(dp[i - 1][j] + 1,dp[i][j - 1] + 1);

插入同理，那么状态转移方程就是dp[i][j] = min(d[i - 1][j - 1] + (s1[i - 1] != s2[j - 1]),min(dp[i - 1][j],dp[i][j - 1]) + 1);

至于初始化，dp[i][0] = i,dp[0][i] = i;其实我还是不太懂0.0

#include <bits/stdc++.h>
using namespace std;

int dp[1111][1111];

int main()
{
    string s1,s2;
    cin >> s1 >> s2;
    memset(dp,0,sizeof(dp));
    for(int i = 0;i <= s1.size();i++)
        dp[i][0] = i;
    for(int i = 0;i <= s2.size();i++)
        dp[0][i] = i;
    for(int i = 1;i <= s1.size();i++)
    {
        for(int j = 1;j <= s2.size();j++)
        {
            dp[i][j] = min(dp[i - 1][j - 1] + (s1[i - 1] != s2[j - 1]),min(dp[i - 1][j] + 1,dp[i][j - 1] + 1));
        }
    }
    cout << dp[s1.size()][s2.size()] << endl;
}

然后做了一道裸的多重背包，不过我第一次写的最裸的算法T了，请教了一下朋友，说是要用二进制的思想进行优化。

二进制的思想就是把多重背包做成01背包来搞，假设你的物品数是9，那么就可以拆分成1，2，4，2的物品，也就是根据二进制来拆出来的物品最后一定能产生所有可能的组合别人的讲解在此

题目在此

#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n,m,w[11111],p[11111],dp[55555],maxx = 0,cnt = 0;
    int a,b,d;
    scanf("%d %d",&n,&m);
    memset(dp,0,sizeof(dp));
    for(int i = 0;i < n;i++)
    {
        scanf("%d %d %d",&a,&b,&d);
        for(int j = 1;j <= d;j <<= 1)
        {
            w[cnt] = a * j,p[cnt++] = b * j;
            d -= j;
        }
        if(d)
        {
            w[cnt] = a * d,p[cnt++] = b * d;
        }
    }
    for(int i = 0;i < cnt;i++)
    {
        for(int j = m;j >= w[i];j--)
        {
            dp[j] = max(dp[j],dp[j - w[i]] + p[i]);
        }
    }
    cout << dp[m] << endl;
}

最大子段和，在一个整数序列中截取一段，使其和最大题目,用dp[i]表示以a[i]为起始位置的最大和。

#include <bits/stdc++.h>
#define ll long long
using namespace std;

int main()
{
    ll n,a[50005],dp[50005],maxx = 0;
    memset(dp,0,sizeof(dp));
    cin >> n;
    for(int i = 0;i < n;i++)
    {
        scanf("%lld",&a[i]);
    }
    for(int j = 0;j < n;j++)
    {
        dp[j] = max(a[j],dp[j - 1] + a[j]);
        maxx = max(maxx,dp[j]);
    }
    cout << maxx << endl;
}

于是我又做了一道求一个数列中两个最大不重叠子段和。题目，思想也是一样，枚举中间的点，把其分成两个数列就行。>

#include <iostream>
#include <cstring>
#include <algorithm>
#define ll long long
#define MAXN 100100
using namespace std;

int main()
{
    int t;
    cin >> t;
    while(t--)
    {
        int n,dp1[MAXN],dp2[MAXN],a[MAXN],maxx = -5554555,dp3[MAXN];
        memset(dp1,0,sizeof(dp1));
        memset(dp2,0,sizeof(dp2));
        memset(dp3,0,sizeof(dp3));
        cin >> n;
        for(int i = 0;i < n;i++)
        {
            scanf("%d",&a[i]);
        }
        dp1[0] = a[0],dp2[n - 1] = a[n - 1];
        for(int i = 1;i < n;i++)//tail  
        {
            dp1[i] = max(a[i],dp1[i - 1] + a[i]);
        }
        for(int i = n - 2;i >= 0;i--)//head
        {
            dp2[i] = max(a[i],dp2[i + 1] + a[i]);
            //cout << dp2[i] << " ";
        }
        dp3[n - 1] = dp2[n - 1];
        for(int i = n - 2;i >= 0;i--)
        {
            dp3[i] = max(dp3[i + 1],dp2[i]);
        }
        for(int i = 1;i < n;i++)
        {
            maxx = max(maxx,dp1[i - 1] + dp3[i]);
            //cout << dp1[i] << " " << dp2[i + 1] << endl;
        }
        cout << maxx << endl;
    }
}

dp1代表以i结尾的最大和，dp2代表以i开始的最大和，dp3是i到结尾的最大和，然后枚举就行了
