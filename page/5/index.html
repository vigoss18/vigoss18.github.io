题目链接：Tunnel Warfare

题意是有n个相连的村庄，有3种操作，第一种是操作D，可以把一个村庄摧毁掉，第二种是操作Q，查询包括$Q_i$在内的最长连续区间的长度，第三种是操作R，可以恢复最近被摧毁的一个村庄。

结构体的构造函数

发表于 2016-03-08 | 分类于 ACM |

今天写一个题的时候，写好结构体之后发现死活不过了编译。当时我code是这样的：

struct both
{
    int num;ll a,b;
    both(int q,ll w,ll e)
    {
        num = q,a = w,b = e;
    }
}node[MAXN];

在结构体里写了一个构造函数，可以用来偷懒啊……但是光是这样写是不对的，如果我写了有参构造函数的话，在创建数组的时候成员没有带参数，就会报错。所以还要加上无参构造函数才行。

struct both
{
    int num;ll a,b;
    both(){}
    both(int q,ll w,ll e)
    {
        num = q,a = w,b = e;
    }
}node[MAXN];

这样就好了。
